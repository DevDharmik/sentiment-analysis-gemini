# 🎯 Sentiment Analysis on YouTube Comments using Gemini & YouTube Data API

This project analyzes the **sentiment of comments** from a YouTube video using the **YouTube Data API** and **Gemini (Google AI)**. It automatically classifies comments into **positive**, **neutral**, or **negative** sentiments, helping you understand audience reactions more effectively.

---

## 🚀 Features

- 🔗 Fetches top comments from any public YouTube video using the **YouTube Data API**
- 🤖 Uses **Google Gemini (LLM)** to analyze and classify the sentiment of each comment
- 📊 Outputs results with original comment and corresponding sentiment
- Optional: Displays results in the console or through a **Streamlit web interface**

---

## 🛠️ Tech Stack

- **Python 3.8+**
- **YouTube Data API v3**
- **Gemini API** (Google Generative AI / PaLM)
- Optional: **Streamlit** for web UI
